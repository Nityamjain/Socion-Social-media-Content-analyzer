# Socion - Social Media Content Analyzer

A Python-based content analysis tool for social media creators.

## Overview

Socion helps you analyze your social media content to optimize engagement and readability. Upload your content, and get comprehensive analysis results in JSON format.

## How to Use

1. **Upload Your File**: Use the web interface to upload PDF, DOCX, or image files
2. **Get Analysis**: The system analyzes your content across multiple dimensions
3. **View Results**: See insights in the dashboard
4. **Further Analysis**: Use JSON output for custom processing

## Key Features

-  Multi-format file upload (PDF, DOCX, Images)
-  Comprehensive content analysis
-  JSON-structured output
-  Easy integration for further processing
-  Engagement prediction
-  Sentiment & emotion detection

## Architecture

**Core Engine**: `backend/flow.py` - Coordinates all analysis modules

**Analysis Modules**:
- Sentiment Analysis
- Emotion Detection
- Keyword Extraction
- Readability Analysis
- Engagement Prediction
- Consistency Checking
